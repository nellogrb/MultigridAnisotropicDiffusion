Detailed Description

template<unsigned int VDimension>
class itk::mad::DirectSolver< VDimension >

Implementation of a direct solver to be used on the coarsest grid for problem.

It internally converts the Image and StencilImage objects to vnl objects vectors and matrix in order to use the class vnl_sparse_lu; the conversion of the matrix and its LU decomposition is generated and stored at construction time, and is available for every subsequent step to avoid unnecessary computations.
