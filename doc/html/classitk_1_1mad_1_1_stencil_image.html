Detailed Description

template<class TPixelType, unsigned int VDimension>
class itk::mad::StencilImage< TPixelType, VDimension >

This class is derived from a specialization of the Image class, where the PixelType is defined as a Neighborhood of values: it is however templated in order to mimic an Image class. It is used by the other classes in the ITKMultigridAnisotropicDiffusion, and can been interpreted as something similar to a sparse matrix where every row corresponds to a single pixel (i.e., a Neighborhood). It also expands the base class, containing a list of active offsets; this is useful in order to easily get the neighbors which need to be visited.
